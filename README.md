# MERN Cleanliness Tracker

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) project for reporting and tracking local environmental and cleanliness issues.  

Users can:
- Report and track issues like garbage buildup, broken footpaths, illegal dumping, waterlogging, etc.
- Request cleanup drives and pay small fees for community services (if applicable)
- View their issue history

---

## Features

- Clean and responsive UI with Tailwind CSS  
- Firebase Authentication for secure login/signup  
- Private and protected routes for authorized users  
- Modern app structure with full CRUD functionality  
- Real-time issue tracking  

---

## Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** Firebase
